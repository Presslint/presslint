# Contributing

This project is not ready for broad external contributions yet. The public
contract is still being shaped.

When contributing:

- keep all source contributions Apache-2.0 compatible;
- do not add GPL, AGPL, SSPL, BUSL, or source-available dependencies;
- preserve deterministic output in write paths;
- preserve byte-identical round-trip for unmodified content streams;
- add focused tests for parser, serializer, selector, and action behavior;
- keep private PDFs, filenames, job identifiers, and corpus excerpts out of
  committed files.
